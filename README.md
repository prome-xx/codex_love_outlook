指纹浏览器+outlook邮箱注册codex。。。稳定使用

无批量功能。自用很爽。bitbrowser免费一天可以开50次窗口，开发给自用的。
用指纹浏览器注册的，5天没被杀一个号。协议注册的大概被杀了30%。可能功能仍然不稳定，我自己也用的免费版，一天50次，测试不充分。而且和IP什么的关系大

优点   
1 稳定，稳定，稳定
2 注册登录，各种使用都会回写邮箱的 refresh_token。号不被封，下次可以直接调登录
3 注册 登录 流程分离。目前试下来 指纹浏览器注册 + 协议登录就已经很稳了。当然，测试仍不够充分，并且 openai天天变花样
4 遇到电话号码过不了了，重新换指纹再来一次仍有可能成功
5 自己去找outlook邮箱，3-4分钱一个  邮箱-密码-CLIENT_ID-REFRESH_TOKEN  格式的

软件内有。用的bitbrowser，需要额外安装配置
